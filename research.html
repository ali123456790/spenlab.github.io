Research Interests

Our key focus areas involve protecting data, ensuring privacy, and optimizing performance in modern networked systems.

  • Design secure communication protocols in VANETs

IoT and Smart Grid Security

  • Analyze vulnerabilities in large-scale IoT sensor networks
  • Secure data management in smart grid AMI

Blockchain-based Energy Trading

  • Explore decentralized energy marketplaces for EVs or prosumers
  • Develop privacy-preserving blockchain protocols

Advanced Data Mining & Machine Learning

  • Implement ML algorithms for cyberattack detection
  • Big data mining for proactive threat detection

Next-Generation Network Security

  • Investigate 5G/6G vulnerabilities and robust security measures
  • Secure routing, authentication, and key management
